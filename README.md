# TransferDB

TransferDB is a two-phase CMPE 321 database project about football transfers, clubs, competitions, matches, and role-based workflows.

This repository combines both course submissions in one GitHub-ready repo:

- `project1/`: relational database design and MySQL schema
- `project2/`: Flask and MySQL web application built on the TransferDB database

## Repository Structure

```text

|   |-- README.md

|   `-- schema/
|-- project2/
|   |-- README.md
|   |-- app.py
|   |-- sql/
|   `-- templates/
`-- README.md
```

## Project 1

Project 1 contains the initial database design deliverables and the MySQL schema implementation.

See [`project1/README.md`](project1/README.md) for details and local usage.

## Project 2

Project 2 contains the continuation of the same project as a Flask web application with MySQL persistence, seeded data, role-based dashboards, triggers, and stored procedure support.

See [`project2/README.md`](project2/README.md) for setup and run instructions.
